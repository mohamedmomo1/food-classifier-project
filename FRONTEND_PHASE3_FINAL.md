## 🚀 كيفية التشغيل

### تشغيل الـ Backend:
```bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
python manage.py runserver
```

### تشغيل الـ Frontend:
```bash
cd frontend
npm install
npm run dev
```

- اختر "كوتش" أو "مشترك"
- سجل البيانات

- إذا كنت كوتش، اذهب إلى `/coach`
